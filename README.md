# Running Local LLM's on iPhone

A single-page website exploring why running large language models locally on iPhones matters, how Apple's MLX framework makes it possible, and what the real trade-offs are.

## Where?

[local_llms](https://local-llms-3br.pages.dev/)

## What?

- `index.html` and `style.css` make up the full article, which covers MLX, unified memory, quantization, on-device privacy benefits, hardware drawbacks like battery drain and thermal throttling, and apps like Locally AI that make this accessible today.
- Reflection memo is also included in the repo as well as on the website, as part of assignment #4 for ENGW3302.
